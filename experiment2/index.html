Experiment 2 - Living Impression

Description

This project offers a generative, living impression of this inspiring image:

Technical

Relation to inspring image:
- Color: The sky, grass, trees, and butte colors were sampled from the inspiring image using the Photoshop eyedropper tool.
- Primitives: The trees are rendered with `triangle` primitives.
- Polygons: The background butte is our custom-shaped polygon using `vertex` calls.
- Generativity: Different random seed values will scatter trees around the front of the butte.
- Life: The clouds in the background will respond to the motion of the mouse cursor to stimulate movement and wind.

Reflection

It was interesting creating a project just from an image we found online. There were definitely moments of confusion, especially regarding the creativity aspect, but I think the project ended up looking nice while following all the requirements.
